## Elite Level Account Manager v2.0.0

Launch and manage multiple Roblox accounts simultaneously — now with a fully reimagined glass UI, Anti-AFK, auto-relaunch, and performance limiting.

### What's New
- Complete visual overhaul — real Windows Acrylic blur-behind, tinted glass panels, gold-accented toggles and scrollbars
- Anti-AFK — briefly focuses the window and sends a real keypress to prevent idle-kick, then restores your previous window. Configurable per-account, with usage statistics
- Auto-Relaunch — automatically relaunches an account if it closes unexpectedly, with a safety cap against crash-loops
- Performance / Graphics Limiting — FPS cap and quality override across all launched clients for smoother multi-boxing
- Update Checker — checks for new releases on launch, Discord-style, with a Settings toggle to disable
- More reliable mutex handling — Fix Now now verifies Roblox has fully closed before reholding

### Features
- Multi-account management — add accounts via browser login or cookie import
- Per-account game selection with saved games library
- Anti-AFK and Auto-Relaunch, both toggleable per account
- Performance limiting across all clients
- Launch presets for quick multi-client sessions
- Window layout editor — arrange clients across monitors
- Active clients monitor with per-client close controls
- Roblox Charts dashboard
- Multi-instance — you already know!
- Focus toggle hotkey — expand any Roblox window to fullscreen and restore it to its configured layout when done!
- Automatic update checks, with an opt-out in Settings

### Installation
Download and run `Elite Level Account Manager Setup 2.0.0.exe`

### Requirements
- Windows 10/11 x64
- .NET Framework 4.x (pre-installed on most Windows machines)
